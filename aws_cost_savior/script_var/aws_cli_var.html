Getting Started


At the top of the script are references in the script that you fill in with values of your choice. Examples include: hostname and IP address. Some variables might have more than one so these variables like hostname1, hostname2. The reference table below has the variable name in human-readable form, a description, and the variable name. Backup the script and then edit a copy with your variables and save the script.
